# Grokking Algorithms

Learning algorithms from the book "Grokking Algorithms" by Aditya Bhargava.

## Chapters

| Folder | Topic |
|--------|-------|
| BinarySearch | Binary search implementations |
| SelectionSort | Selection sort algorithm |
| Recursion | Recursion basics and examples |
| Quicksort | Quicksort and divide & conquer |
| HashTables | Hash tables and dictionaries |
| BreadthFirstSearch | BFS and graph basics |
| Dijkstra | Shortest path with weighted graphs |
| GreedyAlgorithms | Greedy approach and set cover |
| DynamicProgramming | DP, knapsack, LCS |
| KNearestNeighbors | KNN and basic ML |

## How to Run

```
python FolderName/filename.py
```

Example:
```
python BinarySearch/binary_search.py
```
